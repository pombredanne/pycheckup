About

Inspired by @holman and his talk Aggressively Probing Ruby Projects, I decided to do a broad analysis of the most popular python repositories on Github. A majority of the analysis was inspired by hopper and adopted for python.

I wrote more about the project and its findings in a blog post.

Colophon

The source for this app is available on Github.

The app is written in python, data is stored in MongoDB (hosted at MongoLab) and uses the map reduce functionality to compute summary statistics. The frontend is made using Django. The graphs are made in d3. It is hosted on Heroku.
